TODO LIST im Sinne TABLE

        Aufgabe 1
        Holen Sie todos von https://jsonplaceholder.typicode.com/todos.
        Zeigen Sie dies in Form einer Tabelle an.

        Aufgabe 2
        Fügen Sie die Möglichkeit hinzu, ein Todo zu löschen.
        Die Id vom zu löschenden Todo wird durch ein Input eingegeben.
        Denken Sie an die Methode deleteRow(index).

        Aufgabe 3
        Benutzer bekommt die Möglichkeit, die Titel von Todos zu editieren
        Die Änderungen werden nur beim Klick auf Enter übernommen, nicht bei jedem Tippen und nicht beim Verlassen des Titel-Feldes.

        Aufgabe 4
        Legen Sie eine Klasse 'Todo' an und prüfen Sie mit Hilfe der Klasse, ob alle Objekte im JSON die gleichen Props haben, wie die Klasse.

        Aufgabe 5
        Leiten Sie von der Klasse 'Todo' eine weitere Klasse 'Ticket' ab, mit Eigenschaften
         - Autor (string, private)
         - Deadline (Date)
         - Zugewiesen (string)
         - getter deadline mit nur Datum und Uhrzeit in der Rückgabe
         - static
