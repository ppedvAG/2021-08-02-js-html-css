        SYNTAX IN HTML-DATEI
        html-Dateien können folgende Inhalte beinhalten:
        - html (alles, was in spitzen Klammer steht)
        - js (was im Tag script steht)
        - css (was im Tag style steht)


        FORMATIERUNG IN HTML-DATEI
        Nicht jede Formatierung in der HTML-Datei wird vom Browser wiederspiegelt:

        - leere Zeichen um die Inhalte herum werden abgeschnitten
        - neue Zeilen ignoriert

        Ein Hinweis auf eine neue Zeile in HTML kann ein Leerzeichen zwischen den Elementen sein (siehe die spans)



        HTML-ATTRIBUTE
        stehen immer beim öffnenden HTML-Tag von einem HTML-Element
        _attr-name_="_attr-wert_"
